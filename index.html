To get the big picture Javaflame

  • Does not require code changes
  • Works locally and offline
  • Captures all argument values
  • Not only for my own code
  • See them all together in a single place
  • Does not care about performance

To use it

  • Download the agent
  • Add to the JVM command line:
  • Example:
  • java -javaagent:javaAgent.jar -jar yourApp.jar

A simple fizz buzz

  • java -javaagent:javaflame.jar -jar fizzBuzz.jar 40

Multithread

Too much data - Filter expressions

  • java -javaagent:javaflame.jar=filter:com.example.cakeFactory -jar ...

Filter expressions

  • Default: Qualified name contains
  • filter:cakeFactory
  • Method matcher
  • filter:com.example.cakeFactory.CakeService#sellCake
  • Conditional match expression
  • filter:(com.example.cakeFactory)

Intercept third party (postgres sql)

  • filter:(com.example.cakeFactory)||(org.postgresql.jdbc.PgConnection#prepareStatement)

Ignore calls between two functions

  • filter:(com.example.cakeFactory.CakeController#buyCake)||(com.example.cakeFactory.NationalCakeRegistryService#printResult)

Intercept calls instead

  • java -javaagent:javaflame.jar -jar fizzBuzz.jar 40
